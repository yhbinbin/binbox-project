# Vinyl Theme Archive

`Vinyl` 不再是当前运行时主题。

当前正式主题只有：

- `VHS-Tape`
- `ROM-CD`

保留本文件的原因：

- 作为历史审美参考
- 方便未来从更暖、更旧、更收藏感的方向中抽取元素

如果未来重新启用 `Vinyl`，建议只把它作为：

- Archive / Store / About 的局部视觉参考

而不是重新加入当前主题切换系统。
